# Сопроводительное письмо

## Системный промпт
Для реализации задачи был использован следующий системный промпт:
> «Ты — исследователь. Если ответ респондента поверхностный — задай ОДИН уточняющий вопрос. Если ответ подробный — поблагодари и заверши».

**Почему такое решение:**
Промпт четко определяет роль модели (исследователь) и задает строгий алгоритм действий (бинарный выбор: уточнить или завершить). Ограничение «ОДИН уточняющий вопрос» предотвращает зацикливание диалога и делает взаимодействие с пользователем лаконичным, что критично для простых веб-инструментов опроса.

## Пояснения по коду

### Архитектура
Проект реализован по принципу разделения ответственности (Separation of Concerns):
- **Frontend**: Статический HTML/JS. Использует `fetch` для взаимодействия с API. Минимум стилей для чистоты интерфейса.
- **Backend**: FastAPI (Python). Выступает в роли прокси-сервера между фронтендом и LLM API, обеспечивая безопасность API-ключа (он не доступен на стороне клиента).
- **Infrastructure**: 
    - **Nginx**: Используется как Reverse Proxy для объединения фронтенда и бэкенда на одном порту (80), что избавляет от проблем с CORS.
    - **Docker**: Все сервисы изолированы в контейнерах, что гарантирует идентичность среды разработки и запуска.

### Технические решения
- **Модель**: Использована модель `openai/gpt-oss-120b` через провайдера Hyperfusion.
- **Переменные окружения**: Все чувствительные данные (`API_KEY`, `URL`) вынесены в файл `.env`, который добавлен в `.gitignore`.
- **Здоровье сервисов**: В `docker-compose` настроен `healthcheck` для бэкенда, чтобы Nginx запускался только после полной готовности API.

## Что можно добавить/улучшить
1. **База данных**: Сейчас ответы не сохраняются. Добавление PostgreSQL позволит собирать статистику и анализировать ответы респондентов.
2. **Валидация**: Добавить проверку длины ввода на фронтенде и бэкенде.
3. **История диалога**: Реализовать хранение контекста беседы (через сессии или БД), чтобы модель помнила предыдущий уточняющий вопрос.
4. **UI/UX**: Использовать современный фреймворк (React/Vue) и библиотеку компонентов (Tailwind CSS) для более профессионального вида.
