# Инструкция по запуску проекта

Простой веб-инструмент для исследования пользовательского опыта с интеграцией LLM.

## Требования
- Docker
- Docker Compose

## Запуск

1. Перейдите в папку с проектом:
   ```bash
   cd project
   ```

2. Создайте файл `.env` в корне проекта и добавьте в него учетные данные которые я прислал в тг

3. Запустите приложение в фоновом режиме:
   ```bash
   docker compose up -d --build
   ```

3. Откройте браузер и перейдите по адресу:
   [http://localhost](http://localhost)

## Остановка
Для остановки всех сервисов выполните:
```bash
docker compose down
```
